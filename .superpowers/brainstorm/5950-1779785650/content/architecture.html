Architecture — OpenFoodMap

Vue d'ensemble du système, 100% open source, tourne sur ton PC

Frontend
Next.js 14
  • Carte Leaflet.js
  • Page profil
  • Formulaire inscription
  • Dark/Light mode
Backend (API Routes)
Next.js + NextAuth
  • OAuth OFF (Keycloak)
  • GET /api/contributors
  • POST /api/contributors
  • POST /api/report
Base de données
PostgreSQL + Prisma
  • Table contributors
  • Table reports
  • Statut modération
  • Localisation choisie
Flux principal
Contributeur OFF → Login OAuth Keycloak → Choix localisation + bio → Profil "en attente" → Modérateur valide → Pin visible sur la carte
Déploiement
docker compose up  →  Next.js sur :3000 + PostgreSQL sur :5432
